Settings prerequisities:

1) Oculus browser setting
- Visit chrome://flags/
- Enable  #webxr-hands - WebXR experiences with hands tracking

2) Quest settings
- Enable "Auto Enable Hands or Controllers" in Quest Settings (Device - Hands and Controllers)
